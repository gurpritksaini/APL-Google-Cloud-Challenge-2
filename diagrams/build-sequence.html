backend/services/gemini_service.py
google-generativeai wrapper, model=gemini-2.0-flash, converts history role "assistant"→"model"

Socratic rules, GRASP_META JSON block, mastery gate instructions
5
backend/services/memory_service.py chroma_service tutor_prompt
query prior concepts → format memory context → inject into Gemini system prompt
3
Backend — Routers + App
Backend
6
backend/routers/health.py
GET /api/health → {status, api_key_configured}. Frontend calls on load.
7
backend/routers/sessions.py chroma_service
POST /api/sessions (create), GET /api/sessions/{id} (retrieve history)
8
backend/routers/chat.py gemini_service memory_service chroma_service
POST /api/chat: retrieve memory → build prompt → call Gemini → parse meta → store turn → return response
9
backend/main.py all routers
FastAPI app + CORS (allow localhost:5173) + include all routers
